# Quick Reference Card

## 🚀 Get Started in 3 Minutes

### Step 1: Setup Database
```bash
mysql -u root
USE payrollmanagement;
SOURCE database_setup.sql;
```

### Step 2: Compile
```bash
cd c:\Users\user\OneDrive\Desktop\Payroll Management System
javac -cp "lib/mysql-connector-j-9.6.0.jar" main.java src/*.java
```

### Step 3: Run
```bash
java -cp ".;lib/mysql-connector-j-9.6.0.jar" main
```

## 🔧 Common Commands

### Check if admin accounts exist:
```sql
SELECT * FROM admin;
```

### Check if worker accounts exist:
```sql
SELECT * FROM useracc;
```

### See all employees:
```sql
SELECT * FROM payroll;
```

### Delete all employees (to start fresh):
```sql
DELETE FROM employees;
```

### Drop and recreate database:
```sql
DROP DATABASE payrollmanagement;
CREATE DATABASE payrollmanagement;
SOURCE database_setup.sql;
```

## 🐛 Troubleshooting

| Problem | Solution |

| "Duplicate ID error" | Restart application |
| "No employees shown" | Register your first employee |
